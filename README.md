## NBA Scrapper

App that can scrapp "3 point field goal attemps per game" from website https://www.basketball-reference.com/

## Usage

IN:<br/>
Enter player name: (Name Surname)<br/>
Luka Doncic<br/>

OUT:<br/>
2018-19 7.1<br/>
2019-20 8.9<br/>
2020-21 8.3<br/>
